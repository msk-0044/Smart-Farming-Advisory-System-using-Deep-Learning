Project Overview

This project offers farmers an advanced Smart Farming Advisory web platform that applies deep learning and machine learning technologies to streamline, strengthen, and modernize farm-level decision-making.

  • Crop suggestions generated based on soil properties and weather conditions.
  • Fertilizer recommendations derived from NPK input values.
  • Disease detection using leaf images analyzed through a deep learning model.
  • Guidance on both organic and chemical prevention methods.
