# Smart Farming Advisory System (Web Application)

The Smart Farming Advisory System is a modern web application designed to assist farmers with intelligent crop planning, fertilizer management, leaf disease detection, and prevention guidance.
Built using HTML, CSS, and JavaScript, the platform provides an intuitive, responsive, and visually refined interface that makes advanced agricultural decision-making simple and accessible.
